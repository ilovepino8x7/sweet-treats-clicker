# Sweet Treats

## About
Sweet Treat is a clicker-style (work-in-progress) game themed around desserts, where the aim is to eat as much cake as possible, using cakes to buy upgrades to earn as many slices as possible.

## Link to Play online
Works best on PC or Mac, but can be played on mobile with a bit of cropping:
<br>
<a href="https://ilovepino56.itch.io/sweet">The online deployment on Itch.io</a>

## Controls
Left click to select or poke with your finger
